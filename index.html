Probability Calculator

This freeCodeCamp project is part of the Scientific Computing with Python course. The objective was building an application which creates a hat containing balls of different colors, each color in a specified number (e.g. 3 yellow, 2 blue and 6 green). We want to draw a given number of balls (e.g. 4) from the hat and there is a group of expected balls (e.g. 1 blue and 2 green) which we attempt to draw. The application performs a large number of experiments counting the drawn balls how many times contains the group of expected balls and calculates the probability of it. The project is made with Python. Show Probability Calculator >>
